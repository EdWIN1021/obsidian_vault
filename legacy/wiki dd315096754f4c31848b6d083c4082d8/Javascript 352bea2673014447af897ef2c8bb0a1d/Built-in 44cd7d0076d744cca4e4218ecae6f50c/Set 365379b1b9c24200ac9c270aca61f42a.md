# Set

Owner: Edwin

## Array

```jsx
const set = new Set(["a", "a", "b", "b", "c"]);

// Set(3) { 'a', 'b', 'c' }
```

## String

```jsx
new Set("aabbc");

// Set(3) { 'a', 'b', 'c' }
```
